# 🎬 Netflix Exploratory Data Analysis

## Overview

This project focuses on performing Exploratory Data Analysis (EDA) on the Netflix Movies and TV Shows dataset. The primary objective was to understand the structure of the dataset, clean and preprocess the data, identify meaningful patterns, and present insights using visualizations.

The project was completed as part of the **AI & Machine Learning Internship** at Pluto Academy and demonstrates practical skills in data analysis using Python.

---

## Project Objectives

- Understand the structure of the Netflix dataset
- Perform data cleaning and preprocessing
- Analyze trends and relationships within the data
- Create meaningful visualizations
- Draw business insights based on the analysis

---

## Dataset

- **Dataset:** Netflix Movies and TV Shows
- **Source:** Kaggle
- **Link:** https://www.kaggle.com/datasets/shivamb/netflix-shows

## Project Workflow

### 1. Data Loading
The dataset was imported into Google Colab using Pandas. Initial exploration included checking the number of rows and columns, data types, missing values, and duplicate records.

### 2. Data Cleaning
Several preprocessing steps were carried out to improve the quality of the dataset:
- Removed duplicate records
- Handled missing values
- Converted data into appropriate formats
- Removed unnecessary columns where required

### 3. Exploratory Data Analysis
The dataset was analyzed to answer important questions, including:
- Distribution of Movies and TV Shows
- Content released over the years
- Most common content ratings
- Country-wise content production
- Popular genres
- Movie duration distribution

### 4. Data Visualization
Different visualization techniques were used to represent the findings, including:
- Bar Charts
- Line Charts
- Histograms
- Pie Charts
- Scatter Plots
- Heatmaps

These visualizations helped identify trends and made the analysis easier to interpret.

---

## Key Findings

Some important observations from the analysis include:

- Netflix has a significantly larger collection of Movies than TV Shows.
- The number of titles released increased rapidly after 2015.
- A small number of countries contribute most of the available content.
- Drama and International content appear frequently across the platform.
- Mature-rated content represents a considerable portion of the catalog.
